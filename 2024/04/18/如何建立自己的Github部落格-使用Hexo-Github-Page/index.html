如何建立自己的Github部落格 使用Hexo+Github Page

網路上目前已經有很多這類的教學了，所以這篇比較像是自己的學習紀錄。

Hexo是什麼？

Hexo是基於Node.js開發的網誌框架，特色主要如下：

  • 一鍵部署
  • 簡單操作
  • 支援MarkDown語法
  • 多種模板可參考，詳情請見https://hexo.io/themes/
